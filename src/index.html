Todo List

Utilizando Bootstrap, y luego de aprender sobre localStorage en javascript, realize esta página que sirve para crear una lista. En ella se puede crear, borrar y calificar como completadas las tareas ingresadas. Todo esto manteniendo los datos, incluso al cerrar la página.

Currency

Al buscar una web para consultar el precio de cambio de diferentes monedas, estas estaban llenas de información que no necesitaba. Por lo cual decidí hacer mi propia versión con la data justa.

Weather App
